Responsive Specifications

Desktop Layout

  • Centered on the screen with a max-width (e.g., 500px).
  • Visible backdrop obscures the background content.
  • Sufficient margin from the viewport edges.

Tablet Adaptations

  • Maintains the desktop layout but with potentially larger touch targets for buttons.

Mobile Adaptations

  • Spans the full width and a significant portion of the height of the screen.
  • Often positioned at the bottom or top of the screen for easier reachability.
  • Internal content scrolls if it overflows the viewport height.

Accessibility Features

Keyboard Navigation

When open, focus is trapped within the modal. Users can only tab through interactive elements inside the dialog.
