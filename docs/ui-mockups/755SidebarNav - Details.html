Description

A vertical navigation component for web dashboards. It contains links to the main sections of the application, a header for branding, and a footer for user-specific actions. It is designed to be responsive, collapsing to an icon-only view on tablets and hiding in an off-canvas menu on mobile.

Design System Alignment

This component heavily relies on the design system's foundation, using tokens for colors (primary for active states, neutrals for background/text), spacing for padding and layout, and typography for link text. It is composed of atomic Link and Icon components.

Responsive Specifications

Mobile Adaptations

  • The sidebar is hidden off-canvas by default.
  • A JavaScript-toggled class (`.sidebar-nav--mobile-visible`) brings it into view as an overlay.
  • The layout is always in the 'expanded' style to show text labels on smaller screens.

Tablet Adaptations

  • The sidebar becomes permanently visible on the left side of the layout.
  • It collapses to a narrow, icon-only 'collapsed' state.
  • Link labels are hidden, and tooltips are shown on hover/focus for accessibility.

Desktop Enhancements

  • The sidebar expands to its full 'expanded' width, showing both icons and text labels.
  • Tooltips for the collapsed state are hidden as they are no longer necessary.

Accessibility Features

Keyboard Support

All navigation links are focusable and can be activated with Enter/Space. The tab order is logical.

Screen Reader Support

The main container is a `

` element with `aria-label="Main Navigation"`. The active link uses `aria-current="page"` to inform screen readers of the current location.

Focus Management

A clear, visible focus state is applied to all interactive elements (`.sidebar-nav__link`). When opened as a mobile overlay, JavaScript would be required to trap focus within the sidebar.

Aria Attributes

  • aria-label="Main Navigation"
  • aria-current="page"

Color Contrast

Text and icon colors for both default and active states meet WCAG 2.1 AA contrast requirements against the dark background.
