Description

A versatile component for handling file uploads, with variants for single generic files and image files with previews. It supports multiple states including idle, uploading, success, and error, and is designed to be fully responsive and accessible.

Design System Alignment

This component utilizes the design system's tokens for colors (primary, semantic, neutrals), spacing, typography, and border radii. It also depends on atomic components like Button, Spinner, and Alert for its composition.
