Description

A data visualization component used on dashboards to display key metrics. It has variants for showing single KPIs, time-series line charts, and ranked lists. It supports loading, error, and data-present states.

Design System Alignment

This organism component is built using the design system's foundational elements. It uses color tokens for backgrounds and semantics, spacing tokens for internal layout, and typography tokens for headers and data. It also directly composes atomic components like Spinner and Alert for its state management.
