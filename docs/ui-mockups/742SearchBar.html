SearchBar Component

A composite input for search queries, combining a text input, icon, and button. It supports default, focus, and loading states and is responsive.

Default State

Focus State

This state is automatically applied when the user clicks into the input field. The `pseudo-focus-within` class is used here for demonstration.

Loading State

This state is applied by adding the `is-loading` class to the main `search-bar` container, typically via JavaScript while waiting for an API response.

Component Variations

Variant Name: "Simple"

The primary and only defined variant. It includes a prefixed icon, a text input, and a submit button. It adapts to screen size, being full-width on mobile and a fixed width on desktop.
