Component Variations

Variant Name : "confirmation"

Used for critical actions like deletion, requiring explicit user confirmation. Typically has a title, a brief message, and two distinct action buttons (e.g., 'Confirm', 'Cancel').

Variant Name : "form-modal"

Contains a form with input fields for data entry, such as adding or editing an item. It includes a body for the form elements and a footer with 'Save' and 'Cancel' buttons.

Variant Name : "info"

A simple variant for displaying informational messages to the user. It typically contains a title, a body of text, and a single 'Close' or 'OK' button.
