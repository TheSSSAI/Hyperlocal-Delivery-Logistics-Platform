Description

A comprehensive form for vendors to add or edit product details, including name, description, price, stock, category, and image. It includes validation states and a saving state.

Design System Alignment

This organism is composed of molecular and atomic components from the design system, such as FormGroup, Button, FileUpload, and Dropdown. It uses design system tokens for spacing, typography, and colors to ensure consistency.

Responsive Specifications

Mobile Adaptations

  • A single-column layout is used for all form fields, ensuring a clear vertical flow on small screens.

Tablet Adaptations

  • Maintains the single-column layout or may adapt to a two-column layout if space permits, but the primary design is mobile-first.

Desktop Enhancements

  • Uses a two-column grid for most fields to make efficient use of space, with the Product Name and Description fields potentially spanning both columns for better readability.
