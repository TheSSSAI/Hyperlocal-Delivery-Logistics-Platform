Description

A FormGroup is a molecular component that encapsulates a Label, a form input (like TextInput), and an optional error message. It manages the layout, spacing, and accessibility associations between these atomic elements, providing a consistent structure for form fields.

Design System Alignment

This component utilizes the design system's spacing scale for vertical rhythm, typography tokens for the label and error message, and semantic color tokens for the error state. It composes the atomic 'Label' and 'TextInput' components.

Responsive Specifications

Mobile Adaptations

  • The FormGroup maintains its vertical stack layout.
  • The component's max-width is removed to allow it to expand to the full width of the screen.

Tablet Adaptations

  • The FormGroup maintains its desktop max-width.

Desktop Enhancements

  • The FormGroup has a defined max-width to ensure readability and consistent form layouts.

Accessibility Features

Keyboard Support

Standard keyboard interaction is inherited from the child input element.

Screen Reader Support

The `label` is programmatically associated with the `input` via the `for` and `id` attributes. The error message is associated with the `input` via the `aria-describedby` attribute.

Focus Management

Focus behavior is handled by the child input element.

Aria Attributes

  • aria-describedby

Color Contrast

The error text color meets WCAG AA contrast ratios against the background.

CSS Implementation

/* FormGroup CSS */ .form-group { margin-bottom: var(--space-24); max-width: 400px; /* Consistent with single input max-width */ } .form-group__error-message { color: var(--color-semantic-error); font-size: var(--font-size-caption); margin-top: var(--space-8); display: none; /* Hidden by default */ } /* Error State */ .form-group--error .label { color: var(--color-semantic-error); } .form-group--error .input { border-color: var(--color-semantic-error); } .form-group--error .input:focus { border-color: var(--color-semantic-error); box-shadow: 0 0 0 3px var(--color-danger-light); } .form-group--error .form-group__error-message { display: block; /* Show the error message when the group has the error class */ } /* Responsive Adaptations */ @media (max-width: 768px) { .form-group { max-width: 100%; /* Allow full width on mobile */ } }
