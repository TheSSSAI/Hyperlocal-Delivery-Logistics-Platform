AppHeader Organism

This is the main application header for vendor and admin dashboards. It is fixed to the top of the viewport and contains primary navigation triggers and utility controls. Below is a placeholder to demonstrate scrolling behavior.

Default State

Standard header with no active notifications.

Notifications Active State

To showcase this state, a class .app-header__notification-bell--active would be added to the bell container, making the red dot visible.

Responsive Behavior

On screens smaller than 768px, the hamburger menu appears, and the profile dropdown is simplified to just an avatar. Resize your browser window to see this effect.

Component Variations

Variant Name : "vendor-dashboard"

This is the only defined variant for the `AppHeader` in the provided specification. It includes a logo/title on the left, and notification and profile dropdown controls on the right. It is responsive and integrates with a sidebar for mobile navigation.
