Component Variations

Variant Name : "confirmation"

A modal used to get explicit user confirmation for an action, especially a destructive one. It typically has a clear question and two opposing actions (e.g., 'Confirm' and 'Cancel').

Variant Name : "form-modal"

A modal that contains a form for creating or editing data. This allows users to perform an action without navigating away from their current context. It usually contains input fields and 'Save'/'Cancel' actions.

Variant Name : "info"

A simple modal used to display information to the user. It typically has a single action button to acknowledge the message (e.g., 'OK' or 'Got it').
