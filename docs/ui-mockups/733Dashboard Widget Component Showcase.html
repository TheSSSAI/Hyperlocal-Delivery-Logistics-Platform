Component Variations

Variant Name: "KPI Card"

Displays a single, key performance indicator. It includes a large value, a label, and a trend indicator. It supports loading (skeleton) and error states.

Variant Name: "Line Chart Widget"

A container for displaying time-series data. This example shows a placeholder SVG to represent where a real charting library would render. It supports loading (skeleton) and error states.

Variant Name: "List Widget"

Displays a simple list of key-value pairs, ideal for showing rankings like 'Top Selling Items' or 'Recent Activity'. It supports loading (spinner) and error states.
