Responsive Specifications

Desktop Layout

  • The Geofencing Tool variant is designed for desktop, featuring side panels for information and controls, allowing for precise mouse-based drawing.
  • Live-tracking and address-picker variants render within constrained containers.

Tablet Adaptations

  • Maintains the mobile layout but with more spacious UI elements and larger tap targets.

Mobile Adaptations

  • Full-screen or near-full-screen map views to maximize screen real estate.
  • Information panels are often presented as bottom sheets to be less intrusive.
  • All interactions are optimized for touch.

Accessibility Features

Keyboard Navigation

Interactive controls like buttons and panels are keyboard navigable. The map canvas itself is not navigable via keyboard.

Screen Reader Support

Key information like ETA, selected address, or zone status is presented in accessible text elements. Map visual data is not directly accessible.

Focus Management

Focus is managed for any modal dialogs or panels that appear over the map.

Aria Attributes

  • aria-label for icon buttons
  • role='status' for real-time updates like ETA changes or signal loss messages.

Color Contrast

WCAG 2.1 AA compliance for all text and interactive elements overlaid on the map.
