Quantity Stepper

States and Variations

Default

The standard interactive state where both increment and decrement are possible.

Minimum Value (Minus Disabled)

When the quantity is at its minimum (e.g., 1), the decrement button is disabled.

Maximum Value (Plus Disabled)

When the quantity reaches the maximum available stock, the increment button is disabled.

Fully Disabled

The entire component can be disabled, for example, when an item is out of stock.

Component Variations

Variant Name : "default"

The standard interactive state where both increment and decrement are possible.

Variant Name : "disabled-minus"

The minus button is disabled, typically when the quantity is at its minimum value (e.g., 1).

Variant Name : "disabled-plus"

The plus button is disabled, typically when the quantity has reached the maximum available stock.
