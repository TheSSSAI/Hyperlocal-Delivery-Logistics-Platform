Component Name

StarRatingDisplay

Component Type

Description

A read-only component for visually displaying a numerical rating using star icons. It supports fractional ratings by showing partially filled stars. It is purely for display and is not interactive.

Design System Alignment

Uses the design system's icon component and semantic color tokens for filled and empty states (--color-semantic-warning for filled, --color-neutral-200 for empty).

Responsive Specifications

Mobile Adaptations

  • The component is size-agnostic and scales based on the font-size or --star-size variable provided in its context. No specific responsive adaptations are needed.

Tablet Adaptations

  • Same as mobile.

Desktop Enhancements

  • Same as mobile.

Accessibility Features

Keyboard Support

Not applicable as the component is not interactive.

Screen Reader Support

The root element has `role="img"` and a descriptive `aria-label` (e.g., "Rated 4.5 out of 5 stars") to convey the rating value to assistive technologies.

Focus Management

Not applicable.

Aria Attributes

  • role="img"
  • aria-label="{rating} out of 5 stars"

Color Contrast

The contrast between the filled star color and the empty star color is not critical for conveying information, as the shape itself does. The aria-label provides the definitive value.

CSS Implementation

/* StarRatingDisplay CSS */ /* Component Base */ .star-rating { position: relative; display: inline-block; vertical-align: middle; } /* Background and Foreground Containers */ .star-rating__stars-background, .star-rating__stars-foreground { display: flex; gap: var(--space-4); /* Gap between stars */ } /* Foreground is an absolute overlay */ .star-rating__stars-foreground { position: absolute; top: 0; left: 0; white-space: nowrap; overflow: hidden; } /* Icon Styling */ .star-rating .icon { font-size: var(--star-size, 20px); /* Default size, can be overridden */ flex-shrink: 0; } .star-rating__stars-background .icon { color: var(--color-neutral-200); /* Empty star color */ } .star-rating__stars-foreground .icon { color: var(--color-semantic-warning); /* Filled star color */ } /* Size Variants */ .star-rating--sm .icon { font-size: 16px; } .star-rating--lg .icon { font-size: 24px; }
