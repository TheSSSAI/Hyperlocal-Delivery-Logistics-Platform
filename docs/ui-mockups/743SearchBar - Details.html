Description

A self-contained input group that allows users to enter search queries. It combines a text input, a search icon, and a submit button. It includes states for default, focus, and loading.

Design System Alignment

Uses design system tokens for colors (primary, neutrals), spacing, border-radius, and typography. It is composed of atomic components: TextInput, Button, and Icon.

Responsive Specifications

Mobile Adaptations

  • The component takes up the full available width (`max-width: 100%`) on screens smaller than 768px.

Tablet Adaptations

  • The component may still be full-width depending on the layout context.

Desktop Enhancements

  • The component has a fixed `max-width` (e.g., 480px) to prevent it from becoming too wide in desktop layouts.

Accessibility Features

Keyboard Support

The entire component is focusable. Users can Tab to the input field, type, and then Tab to the submit button. Pressing Enter within the input field submits the form.

Screen Reader Support

Focus Management

The `:focus-within` pseudo-class provides a clear visual outline around the entire component when any of its interactive children (input, button) are focused.
