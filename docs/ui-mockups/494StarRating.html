Used in forms to collect user feedback. It is built with radio inputs for full keyboard and screen reader accessibility.

Component Variations

Variant Name : interactive

Allows a user to select a rating from 1 to 5. It is built using visually hidden radio buttons and styled labels for maximum accessibility.

Variant Name : read-only

Displays a static rating value, often an average. It supports partial stars to accurately represent decimal values (e.g., 4.7 stars).
