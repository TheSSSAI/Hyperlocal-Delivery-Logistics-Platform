Component Variations

Variant Name : add

The 'Add Product' variant of the form. All fields are empty by default, prompting the user for new input.

Variant Name : edit

The 'Edit Product' variant of the form. All fields are pre-populated with the existing product's data, allowing the vendor to make modifications.
