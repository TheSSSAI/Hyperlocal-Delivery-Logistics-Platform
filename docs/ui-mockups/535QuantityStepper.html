Quantity Stepper Component

Default State

Both increment and decrement buttons are enabled.

Disabled Minus State

Used when the quantity is at its minimum value (e.g., 1 or 0).

Disabled Plus State

Used when the quantity reaches the maximum available stock.

Fully Disabled State

Used when the entire component is non-interactive, such as for an out-of-stock item.

This is a self-contained HTML file. All styles are embedded in the <head>.
