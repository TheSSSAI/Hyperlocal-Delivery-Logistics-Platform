Component Variations

Variant Name : "idle"

The initial state of the wizard, prompting the user to upload a CSV file. It includes a dropzone and a link to download the required template.

Bulk Import Products

Save time by uploading your product catalog from a CSV file. Make sure your file matches our template.

Download CSV Template

Variant Name : "uploading"

A transient state shown immediately after a file is selected, indicating that the file is being transferred to the server.

Uploading File...

Your file is being securely uploaded.

Variant Name : "processing"

A transient state shown after the upload is complete, indicating that the server is validating and importing the data.

Processing Your File

Please wait while we validate and import your products. This may take a few moments.

Variant Name : "complete-with-errors"

A final state indicating that the import process finished, but some rows failed validation. It provides a summary and a clear call-to-action to download a detailed error report.

Import Complete with Errors

Some rows could not be imported.

We successfully imported 25 products, but 5 rows had errors. Download the report to see details and correct your file.

Variant Name : "complete-success"

A final state indicating that the entire CSV file was processed successfully. It provides a summary of created and updated products.

Import Successful

Your catalog has been updated.

7 products were created and 3 products were updated.
