Dashboard

This is the main content area. The sidebar is responsive and will change based on the viewport width.

  • Desktop (>1024px): The sidebar is expanded with icons and text.
  • Tablet (768px - 1024px): The sidebar collapses to show icons only, with tooltips on hover/focus.
  • Mobile (<768px): The sidebar is hidden. Click the hamburger icon (top-left) to toggle the off-canvas menu.

Component Variations

Variant Name : "admin"

The primary variant used for the Admin and Vendor dashboards. It features a dark background with light text and icons, and a primary color accent for the active navigation item.
