FormGroup Component

A layout component that groups a label, an input, and an optional error message for consistent spacing and accessibility.

Default State

The standard appearance of a FormGroup, containing a label and a form input. The error message element is present in the DOM but is hidden.

This is a hidden error message.

Error State

By adding the .has-error class to the main .form-group container, the error message becomes visible and the input's border changes color. The aria-describedby and aria-invalid attributes are crucial for accessibility.

Please enter a valid email address.

Other Variations

The FormGroup can wrap different types of form inputs, like textareas.

This is a hidden error message.

Component Variations

Variant Name: Default

Standard usage for a single form field with its label.

Variant Name: With Error

Used to display a validation error. The .has-error class is added to the .form-group container, and the input has aria-invalid and aria-describedby attributes.

The GST number you entered is not valid.
