Component Variations

Variant Name : "Default"

The standard state of the FormGroup, used for collecting user input without any validation errors. It includes a label and a text input.

Variant Name : "Error"

The error state is activated by adding the `.form-group--error` class. This makes the error message visible and applies error styling to the input and label, providing clear feedback to the user. The `aria-describedby` attribute is crucial for accessibility.
