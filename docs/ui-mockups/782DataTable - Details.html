Description

A versatile data table for displaying sets of data with support for sorting, filtering indicators, pagination, and various states like loading, empty, and error. It is designed to be responsive, with horizontal scrolling on smaller viewports.

Design System Alignment

The DataTable extensively uses the design system's color palette for text, backgrounds, and borders; typography for headers and cell content; and spacing scale for padding and layout. It is built to contain atomic components like Checkbox, Spinner, and Badge.

Responsive Specifications

Mobile Adaptations

  • The table wrapper becomes horizontally scrollable on viewports smaller than its `min-width` (e.g., < 768px).
  • Pagination controls may simplify to only 'Previous' and 'Next' buttons to save space.

Tablet Adaptations

  • The table may still require horizontal scrolling depending on the number of columns and the viewport width.

Desktop Enhancements

  • All columns are visible without scrolling on standard desktop viewports.
  • Hover effects on rows and sortable headers are primarily for mouse-based interaction.

Accessibility Features

Keyboard Support

The table itself is not interactive, but all nested controls (checkboxes, buttons, pagination) are focusable and operable via keyboard.

Aria Attributes

All text and background color combinations adhere to WCAG 2.1 AA contrast ratios.
