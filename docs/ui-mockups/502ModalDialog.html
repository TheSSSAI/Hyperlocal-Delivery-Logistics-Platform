Component Variations

Variant Name: "confirmation"

Used to confirm a critical or destructive user action, like deletion. Typically includes a secondary and a primary (often destructive) action button.

Variant Name: "form-modal"

Used to collect focused user input without navigating to a new page. The body contains form elements, and the footer contains submission actions.

Variant Name: "info"

Used to present important information to the user that must be acknowledged. Typically contains a single action button to dismiss the modal.
