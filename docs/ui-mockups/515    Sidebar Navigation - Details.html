Responsive Specifications

Desktop Layout

  • Full-width sidebar with icons and text labels.
  • Supports both expanded and collapsed states, user-toggleable.

Tablet Adaptations

  • Defaults to the collapsed (icon-only) state to maximize content area.
  • The toggle button is still available to expand if needed.

Mobile Adaptations

  • Sidebar is hidden off-canvas by default.
  • A 'hamburger' menu button in the main header is used to toggle the visibility of the sidebar, which slides in from the left.

Accessibility Features

Keyboard Navigation

Complete tab order through the logo, all navigation links, the collapse toggle, and the user profile/logout link.

Screen Reader Support

Uses a

landmark. Each link has a clear text label. In the collapsed state, links use aria-label to announce their function.

Focus Management

Visible focus indicators on all interactive elements. The active link is marked with `aria-current='page'`.

Aria Attributes

  • aria-label
  • aria-expanded
  • aria-current
  • role='navigation'

Color Contrast

All text and icons meet WCAG 2.1 AA contrast requirements against the sidebar background.
