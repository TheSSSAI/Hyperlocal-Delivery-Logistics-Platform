Description

A fundamental form component that groups a label, an input field, and optional help text or an error message. It ensures proper spacing, accessibility, and state management for a single form entry.

Design System Alignment

This component combines the atomic 'Label' and 'Input' components, and uses 'Alert' styling for error states. It utilizes the design system's typography, spacing, and color tokens for a consistent look and feel.

Responsive Specifications

Mobile Adaptations

  • The component naturally stacks vertically, which is ideal for mobile screens.
  • Font sizes for labels and help text remain readable on small devices.

Tablet Adaptations

  • Maintains the vertical stacking layout.

Desktop Enhancements

  • Can be placed within multi-column grid layouts for more complex forms.
