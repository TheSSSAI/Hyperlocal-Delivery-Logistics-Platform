FileUpload Component

Variant Name: Single File Upload

For uploading generic documents like CSV, PDF, etc. Shows file metadata and progress.

Idle State

The default state, prompting the user to select a file.

Uploading State

Shows an indeterminate progress bar while the file is being uploaded.

Success State

Confirms the file has been uploaded successfully.

vendor_catalog.csv Upload complete

Error State

Indicates that the upload failed and provides an error message.

Invalid file type.

Variant Name: Image Upload with Preview

A specialized uploader for images that displays a thumbnail preview during and after the upload process.

Idle State

The default state for the image uploader.

Uploading State

Shows a dimmed preview of the image with a spinner overlay.

product-image.jpg

Success State

Shows the final image preview with an action to remove it.

Error State

Indicates a failure and allows the user to retry.

Upload Failed
File is too large.
