Component Variations

Variant Name: "grid-view"

Used for desktop and tablet layouts where horizontal space is plentiful. Displays products in a multi-column grid. The component uses `flex-direction: column` and a larger aspect ratio for the image.

Variant Name: "list-view"

The mobile-first default. Used for single-column layouts on smaller screens. Displays the product with the image on the left and content on the right (`flex-direction: row`) for better scannability in a vertical list.
