Page Content

This area represents the main content of the dashboard. The sidebar will overlay this on mobile, and push it to the side on larger screens.

Resize the browser window to see the responsive behavior:

  • < 768px (Mobile): Off-canvas menu activated by the toggle.
  • 768px - 1024px (Tablet): Icon-only sidebar that expands on hover.
  • > 1024px (Desktop): Fully expanded sidebar.
