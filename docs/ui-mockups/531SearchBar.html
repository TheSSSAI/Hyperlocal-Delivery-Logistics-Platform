Molecular Component: SearchBar

Simple SearchBar

A standard search input with a submit/search icon. A clear button appears when text is entered.

Default State

Focus State

The entire component gets a focus ring for better keyboard navigation context.

With Value State

When the user types, the clear button appears.

Loading State

While a search query is being processed, a spinner replaces the clear button.

SearchBar with Filters

A variant that includes a button to trigger filter options. This is common in complex discovery views.

Responsive Collapsible SearchBar

On mobile viewports, this variant collapses to a single icon. When focused or tapped, it expands. On desktop, it remains expanded. Note: In this mockup, click inside the search bar area to see the expand effect.

Component Variations

Variant Name : "Simple"

The default search bar with an input, search icon, and a clear button that appears when there is text.

Variant Name : "With Filters"

Includes an additional button, typically placed next to the search input, to open a modal or panel with advanced filtering options.
