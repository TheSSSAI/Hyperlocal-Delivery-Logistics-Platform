Modal Dialog Component

This page demonstrates the different variants and states of the Modal Dialog component. Modals are used to display critical information, forms, or confirmation prompts without leaving the current page.

Information

This is an informational modal. It's used to convey important messages to the user that require acknowledgment before they can proceed.

Screen Variations

Variant Name: "Info Modal"

Used for displaying important information that requires user acknowledgement. Typically has a single primary action button.

Variant Name: "Confirmation Modal"

Used to confirm a user's intent, especially for destructive or irreversible actions. Typically includes a secondary 'Cancel' button and a primary action button, which may be styled as 'danger'.

Variant Name: "Form Modal"

Used to display a form for creating or editing data without leaving the current page context. The footer contains 'Save' and 'Cancel' actions.
