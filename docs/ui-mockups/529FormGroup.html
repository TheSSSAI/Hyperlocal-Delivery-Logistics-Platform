FormGroup Component

The FormGroup component orchestrates the relationship between a label, an input, and an optional error message, ensuring consistent spacing and accessibility.

Component Variations

Default

Standard grouping of a label and an input field. Used for most simple data entry tasks. Provides vertical spacing and accessibility linkage.

Error

Used when form validation fails for a specific field. It includes a visible error message linked to the input via ARIA attributes, and the input field is styled to indicate an error.
