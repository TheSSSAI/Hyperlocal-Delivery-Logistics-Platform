OTP Input

A molecular component for entering verification codes.

Component Variations

Variant Name : "Default State"

The default appearance of the OTP input fields.

The appearance when a user selects an input field. The border highlights in the primary color.
Focus State (Click a field)

Variant Name : "Filled State"

The appearance when all fields are filled with digits.
Filled State

The appearance when the submitted OTP is incorrect. Borders and text turn to the error color.
