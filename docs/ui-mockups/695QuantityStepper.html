Molecular Component: QuantityStepper

A control for incrementing or decrementing a numerical value, typically used in shopping carts.

Component Variations

Default State

The standard interactive state where the user can increase or decrease the quantity.

Product Quantity

Disabled Minus State

Occurs when the quantity is at its minimum value (e.g., 1). The 'Decrease' button is disabled.

Product Quantity (Minimum)

Disabled Plus State

Occurs when the quantity reaches the maximum available stock. The 'Increase' button is disabled.

Product Quantity (Maximum Stock Reached)
