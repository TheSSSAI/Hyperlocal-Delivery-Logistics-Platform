Responsive Specifications

Desktop Layout

  • Grid layout displaying multiple map variants side-by-side for comparison.
  • Control panels for admin tools are docked to the side or bottom of the map view.

Tablet Adaptations

  • Grid layout adjusts to two columns.
  • Ensures touch targets for map controls and UI elements are sufficiently large.

Mobile Adaptations

  • Single-column stacked layout for variant cards.
  • Control panels are typically presented as bottom sheets to maximize map visibility.

Accessibility Features

Keyboard Navigation

Focus order is logical, moving from control panels to map controls.

Screen Reader Support

Key information (e.g., selected address, ETA) is available to screen readers. Map controls have descriptive ARIA labels.

Focus Management

Clear focus indicators on all interactive elements (buttons, inputs, map controls).

Aria Attributes

  • aria-label
  • aria-describedby
  • role attributes

Color Contrast

WCAG 2.1 AA compliance for all text overlays and UI controls on the map.
