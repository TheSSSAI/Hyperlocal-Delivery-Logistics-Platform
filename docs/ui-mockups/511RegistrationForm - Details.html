Description

A comprehensive, multi-step or single-page form for onboarding different user types (Customer, Vendor, Rider). It combines various molecular components like FormGroup, FileUpload, and OTPInput to capture all necessary information.

Design System Alignment

This organism is a composition of molecular and atomic components from the design system. It uses FormGroup for structure, Button for actions, and FileUpload for document submission, all inheriting the foundational styles and tokens.

Responsive Specifications

Mobile Adaptations

  • The form maintains a single-column layout for all fields, ensuring a clear and simple flow on narrow screens.
  • Padding is slightly reduced to maximize content space on smaller devices.

Tablet Adaptations

  • Maintains the single-column layout, which is robust and easy to use on touch devices.

Desktop Enhancements

  • The form is centered with a max-width for optimal readability on larger screens.
