Dashboard Content

This is the main content area. On desktop, you can use the toggle button to expand and collapse the sidebar. On tablet, it defaults to collapsed. On mobile, it's hidden and can be shown by a menu button in the header.

Component Variations

Variant Name : "vendor"

Displays navigation links relevant to the Vendor role, such as Dashboard, Orders, Products, and Reports.

Variant Name : "admin"

Displays navigation links relevant to the Administrator role, including all vendor links plus User Management and platform-wide Settings.
