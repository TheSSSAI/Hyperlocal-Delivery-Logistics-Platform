Molecular Component: FileUpload

Single File Upload

Used for uploading generic documents like PDFs, CSVs, etc.

Idle State

The default state, ready for file selection.

Uploading State

Shown while the file is being uploaded to the server.

Success State

Shown after a file has been successfully uploaded.

Image Upload with Preview

Specialized variant for image files, showing a thumbnail preview.

Idle State

The default state for image upload.

Uploading State

Shown during image upload.

Success State

Shows a preview thumbnail of the uploaded image.
