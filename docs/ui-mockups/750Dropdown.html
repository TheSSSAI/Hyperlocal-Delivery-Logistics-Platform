Molecular Component: Dropdown

A custom select component for choosing one or more options from a list.

Single-Select Dropdown

Allows the user to select one option from a list.

  • Appetizers
  • Main Courses
  • Desserts
  • Beverages

Multi-Select Dropdown

Allows the user to select multiple options.

  • Lettuce
  • Tomatoes
  • Onions
  • Cucumbers

Component Variations

Variant Name: Single-Select

The standard dropdown where a user can only select one option from the list. The selected value replaces the placeholder in the trigger.

Variant Name: Multi-Select

Allows users to select multiple options from the list. The trigger typically displays a count of selected items rather than the items themselves.
