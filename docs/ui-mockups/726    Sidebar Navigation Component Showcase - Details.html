Responsive Specifications

Desktop Layout

  • Sidebar is permanently visible on the left side in an 'expanded' state, showing both icons and text labels.
  • Main content area occupies the remaining screen width.

Tablet Adaptations

  • Sidebar collapses to an icon-only 'collapsed' state to maximize content area.
  • Tooltips appear on hover to show the link name.

Mobile Adaptations

  • Sidebar is hidden by default (off-canvas).
  • A hamburger icon in the header toggles the visibility of the sidebar, which slides in over the main content.
