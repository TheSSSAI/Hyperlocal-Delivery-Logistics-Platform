Molecular Component: StarRatingDisplay

A read-only component for visually displaying a numerical rating using star icons. It supports fractional ratings by showing partially filled stars.

Examples

4.5 Stars (width: 90%)
3.0 Stars (width: 60%)
5.0 Stars (width: 100%)
0 Stars (width: 0%)
3.7 Stars (width: 74%)

Size Variants

Small (.star-rating--sm)
Default (no size class)
Large (.star-rating--lg)

Component Variations

Variant Name : "read-only"

The default and only variant. This component is for display purposes only and does not accept user input. It's used to show an existing rating value.
