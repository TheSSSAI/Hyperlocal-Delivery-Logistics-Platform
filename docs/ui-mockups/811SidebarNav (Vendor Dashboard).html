Dashboard

This is the main content area. The sidebar provides navigation to different sections of the vendor dashboard.

On desktop, the sidebar is expanded by default. On tablet, it's collapsed. On mobile, it's hidden and can be opened with the hamburger menu icon.

Component Variations

Expanded State (Default on Desktop)

The sidebar shows both icons and text labels, providing clear navigation cues. This is the default state for larger screens.
