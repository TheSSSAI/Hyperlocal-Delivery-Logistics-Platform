Responsive Specifications

Desktop Layout

  • The form is presented within a card or modal with a max-width (e.g., 600px) and centered.
  • A two-column grid can be used for related fields on wider screens (e.g., Price and Stock).

Tablet Adaptations

  • Maintains the desktop layout but ensures touch targets are sufficiently large.

Mobile Adaptations

  • The form takes the full width of the screen.
  • All fields are stacked vertically in a single column to ensure readability and ease of use.
