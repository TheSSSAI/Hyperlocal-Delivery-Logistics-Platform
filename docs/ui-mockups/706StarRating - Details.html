Description

A molecular component for displaying or capturing a 1-to-5 star rating. It supports an interactive mode for user input and a read-only mode for displaying an existing rating, including partial stars.

Design System Alignment

This component is composed of the atomic 'Icon' component. It uses design system tokens for color (semantic warning for filled stars, neutrals for empty), spacing (for touch targets), and transitions.

Responsive Specifications

Mobile Adaptations

  • Ensures button-like stars have a minimum touch target of 44x44px for easy tapping.
  • Component size can be controlled via the `--star-size` custom property for different contexts.

Tablet Adaptations

  • Maintains mobile sizing, which is also appropriate for tablet touch interfaces.

Desktop Enhancements

  • Includes hover states for the interactive variant to provide clear feedback.

Accessibility Features

Keyboard Support

The interactive variant is keyboard navigable using Tab and arrow keys, as it is built on native radio buttons. Spacebar selects a rating.
