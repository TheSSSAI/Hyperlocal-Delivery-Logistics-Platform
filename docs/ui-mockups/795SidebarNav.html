Sidebar Navigation Component

This page demonstrates the responsive SidebarNav organism. The sidebar's behavior changes based on the viewport width.

  • Desktop (> 1024px): The sidebar is expanded by default and can be collapsed to an icon-only view using the toggle at the bottom.
  • Tablet (769px - 1024px): The sidebar is collapsed by default to save space.
  • Mobile (<= 768px): The sidebar is hidden off-canvas. A hamburger menu icon appears at the top-left to toggle its visibility as an overlay.

Use the toggle button in the sidebar footer (on desktop) or the hamburger icon (on mobile, resize your browser) to see the states in action.

Component Variations
